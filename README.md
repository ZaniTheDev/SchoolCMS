# SchoolCMS

SchoolCMS is a school management system built with Next.js, Prisma, and PostgreSQL.

It includes authentication, user management, and basic post handling for school-related data.

The project is structured to be scalable and can be extended with features like assignments, grading, and admin dashboards.

## Tech Stack

- Next.js
- Prisma
- PostgreSQL
- NextAuth / JWT

## Purpose

Built as a learning project to understand full-stack development, authentication, and backend architecture.
